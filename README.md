# clean-blog

**The official repo of Clean Blog**

This repository is designed to store project files, as well as in cases of critical crashes for the backward version.

**Perimities:**

- Registration/Authorization with JWT token's
- Validation of data (in registration and authorization)
- The laying out of the article and their modifications
- Castomization of profile (a standard avatar change is in development)
- Comments under the article(in development)
